Smart ID デジタル名刺とは？

Smart ID デジタル名刺は、スマートフォンで簡単に名刺生成が可能なプラットフォームです。
強固なセキュリティであるブロックチェーン技術を活用し、信頼性のある無限の繋がりを構築する為、ビジネス拡大を加速させます。

Smart ID の 特徴

1.名刺を
簡単作成

2.QRコードで
瞬時に名刺交換

3.データセキュリティ

4.無限のつながりの構築

1.名刺の作成

トップ画面

会員登録を行う

１枚目の名刺を作成する

名刺一覧に保存する


簡単にプロフィール作成が可能で、
部署変更や転職時もすぐに変更できます。


2.QRコードで名刺交換

相手にアプリを
立ち上げてもらう

相手がQRコードを読み取り、
名刺交換する

受け取った名刺を確認する

名刺の詳細を確認する


例えばリモート中でも、QRコードで瞬時に名刺交換が出来ます。

3.データセキュリティー

相手が自分の名刺を
「ブロックチェーンに保存」

名刺交換履歴から
「ブロックチェーン
確認リンク」をタップ

該当のブロックチェーンへ
確認処理が行われる

ブロックチェーン認証済み


ブロックチェーン技術を使用している為
他者による情報の改ざんが検出され、安心です。

4.繋がりの構築

名刺交換履歴から
掲示板リンクをタップ

コミュニティーの掲示板で
対話が可能


名刺交換が増えてブロックチェーンに刻まれると
信頼性情報が追加され、未知の相手とも繋がりが可能です。
